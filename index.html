Paper Recommender

This repository contains a working Python prototype that fetches real research paper metadata from Crossref and recommends papers in the terminal.

How to run

C:/Users/gaiya/AppData/Local/Programs/Python/Python313/python.exe paper_recommenderss.py "machine learning"

Or run without arguments to type a search query interactively.

Interactive commands

  • y / yes — accept
  • n / no — reject
  • s / save / later — save for later
  • l / list — reading list
  • p / profile — feedback summary
  • q / quit — exit

Offline mode

If you need a local fallback, run:

C:/Users/gaiya/AppData/Local/Programs/Python/Python313/python.exe paper_recommenderss.py --local "machine learning"

The default mode uses live Crossref data without placeholders.
